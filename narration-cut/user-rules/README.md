# 使用者偏好

AI 審核時參考的個人偏好設定。

## 檔案清單

| 檔案 | 類型 | 說明 |
|------|------|------|

## AI 審核順序（依優先順序）

1. **靜音 >1s** → 建議刪（依 1 秒為單位拆分）
2. **殘句** → 刪（話說一半 + 靜音）
3. **重複句** → 刪短的（開頭 ≥5 字相同）
4. **句內重複** → 刪 A+中間（A+中間+A 模式）
5. **卡頓詞** → 刪前面（那個那個、就是就是）
6. **重說糾正** → 刪前面（部分重複、否定糾正、詞被打斷）
7. **語氣詞** → 標記待人工確認（嗯啊呃）

## 核心原則

**刪前保後**：後面說的通常比較完整，刪除前面、保留後面。
