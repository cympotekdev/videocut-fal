# 靜音段處理

## 閾值規則

| 靜音長度 | 處理方式 |
|---------|---------|
| ≤ 0.5 秒 | **忽略** — 自然停頓 |
| 0.5-1 秒 | **可選刪除** — 句間停頓 |
| > 1 秒 | **建議刪除** — 明顯卡頓或展示畫面 |

## 輸出格式

**整段標記，不拆分**

範例：3.2 秒靜音 → 輸出 1 條
```
| 64-66 | 12.86-15.80 | 靜音3.2s | | 刪 |
```

使用者在審核網頁中可以取消不想刪的。

## 特殊情況

### 長靜音
連續 5s 以上的靜音整段標記，預選刪除：
```
| 323-371 | 71.38-131.38 | 靜音60s | | 刪 |
```

### 開頭靜音
影片開頭的靜音一定要刪：
```
| 0 | 0.00-1.00 | 靜音1s | 開頭靜音 | 刪 |
```
