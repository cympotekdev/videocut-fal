# 語氣詞偵測

## 語氣詞清單

```javascript
const fillerWords = ['嗯', '啊', '哎', '誒', '呃', '額', '唉', '哦', '噢', '呀', '欸'];
```

## 刪除邊界

```
錯誤：刪語氣詞的時間戳 (語氣詞.start - 語氣詞.end)
      → 可能刪到前面字的尾音

正確：從前一個字的 end 到後一個字的 start
      → (前字.end - 後字.start)
```

## 使用者偏好

保留適量的「嗯」作為過渡，不要全刪。
